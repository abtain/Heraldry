Yadis Conformance

This is currently only a proposal, not adopted by the community.

This page is supposed to evolve into a Yadis conformance suite. Developers and technology evaluators will be able to use the tests on this page to determine whether a product is conformant with the Yadis specifications. The actual tests are developed by the community, and expect to grow over time.
